# My Profile Website

Personal portfolio website for Hlay Aliotte.

This project presents my professional background in IT administration, technical support, computer science, and academic development in a more complete way than a traditional resume alone.

## Overview

The site is a custom static portfolio built with:

- HTML
- CSS
- JavaScript

It includes:

- a hero section with professional introduction
- an About section that explains technical and service-oriented strengths
- detailed work experience
- selected projects
- education, coursework, and technical strengths
- contact information
- downloadable resume and transcript links

## Project Structure

- `index.html` - main website structure and content
- `styles.css` - visual design, layout, theming, motion, and responsive behavior
- `script.js` - theme toggle, reveal behavior, and active navigation tracking
- `docs/` - supporting documents linked from the website
- `high_tech_profile_website_blueprint.md` - planning and content blueprint used during development

## Running Locally

This is a static website, so no build step is required.

You can open `index.html` directly in a browser, or serve the folder locally with any simple static server.

## Notes

- The repository is currently private.
- Sensitive academic information such as the degree audit is intentionally excluded from version control.
- The content and styling were refined to support both dark mode and light mode with stronger readability and accessible contrast.
